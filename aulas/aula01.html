O que é JSX?

JSX é uma extensão da sintaxe do JavaScript que permite escrever HTML dentro do JavaScript.

Ele é usado principalmente com o React para definir a estrutura da interface do usuário.

JSX é compilado para chamadas `React.createElement()`, tornando a sintaxe mais intuitiva.

Considere esta declaração de variável:


const element = 
Hello, world!
;

Este código parece HTML, mas é JSX!

JSX permite escrever código declarativo e reutilizável em React.

O que acontece por trás?

O JSX acima é transformado em:


const element = React.createElement("h1", null, "Hello, world!");

Isso significa que JSX não é obrigatório, mas facilita muito a escrita de interfaces.

Incorporando Expressões em JSX

JSX permite incorporar expressões JavaScript dentro do HTML.

Isso torna o código mais dinâmico e interativo.

Exemplo de Expressão em JSX


const name = "Nator";
const element = 
Olá, {name}!
;

Aqui, `{name}` é uma expressão JavaScript embutida dentro do JSX.

Exemplo com Função


function formatName(user) {
    return user.firstName + " " + user.lastName;
}

const user = { firstName: "Nator", lastName: "Júnior" };

const element = 
Olá, {formatName(user)}!
;

Chamamos uma função diretamente dentro do JSX.
