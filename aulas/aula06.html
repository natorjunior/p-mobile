Interface Interativa


const TaskApp = () => {
  const [tasks, setTasks] = useState([
    { id: 1, text: 'Estudar React Native', done: false },
    { id: 2, text: 'Fazer exercícios', done: false }
  ]);

  const toggleTask = (id) => {
    setTasks(tasks.map(task =>
      task.id === id ? { ...task, done: !task.done } : task
    ));
  };

  return (
    <View style={styles.container}>
      <Text style={styles.title}>Minha Lista de Tarefas</Text>
      {
        tasks.map(task => (
          <TouchableOpacity
            key={task.id}
            onPress={() => toggleTask(task.id)}
            style={styles.task}
          >
            <Text style={[
              styles.taskText,
              task.done && styles.taskDone
            ]}>
              {task.done ? '✅' : '⭕'} {task.text}
            </Text>
          </TouchableOpacity>
          )
        )
      }
    </View>
  );
};

O que é useEffect?

É como um "assistente" que nos ajuda a executar ações extras em nossos componentes:

  • 🔄 Executa código quando algo muda
  • 🎬 Roda quando o componente aparece na tela
  • 🧹 Limpa quando o componente some da tela

Exemplo Básico do useEffect

Atividade 1: Mensagem do Dia

Crie um componente que:

  • 1. Comece com a mensagem "Carregando..."
  • 2. Use useEffect para mudar para "Bom dia!" após 2 segundos
  • 3. Adicione um emoji ☀️ no final da mensagem
  • 4. Use o setTimeout para contagem

Solução: Mensagem do Dia


          import React, { useState, useEffect } from 'react';
          import { View, Text, StyleSheet } from 'react-native';

          const MensagemDoDia = () => {
            const [mensagem, setMensagem] = useState('Carregando...');

            useEffect(() => {
              // Espera 2 segundos e muda a mensagem
              setTimeout(() => {
                setMensagem('Bom dia! ☀️');
              }, 2000);
            }, []);

            return (
              <View style={styles.container}>
                <Text style={styles.texto}>{mensagem}</Text>
              </View>
            );
          };

          const styles = StyleSheet.create({
            container: {
              padding: 20,
              alignItems: 'center',
            },
            texto: {
              fontSize: 24,
              color: '#2c3e50',
            },
          });

useEffect com Dependências

O useEffect pode "observar" mudanças em valores: 


            const [nome, setNome] = useState('');

            useEffect(() => {
              // Roda sempre que 'nome' mudar
              console.log('O nome mudou para:', nome);
            }, [nome]); // nome está na lista de dependências

Atividade 2: Contador com Mensagem

Crie um contador que:

  • 1. Tenha um botão para aumentar o número
  • 2. Use useEffect para mostrar "Número par!" quando o contador for par
  • 3. Mostre "Número ímpar!" quando for ímpar

Solução: Contador com Mensagem


          import React, { useState, useEffect } from 'react-native';
          import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';

          const ContadorMensagem = () => {
            const [contador, setContador] = useState(0);
            const [mensagem, setMensagem] = useState('');

            useEffect(() => {
              // Verifica se é par ou ímpar
              if (contador % 2 === 0) {
                setMensagem('Número par! 👍');
              } else {
                setMensagem('Número ímpar! 👆');
              }
            }, [contador]); // Roda quando contador muda

            return (
              <View style={styles.container}>
                <Text style={styles.numero}>{contador}</Text>
                <Text style={styles.mensagem}>{mensagem}</Text>
                <TouchableOpacity 
                  style={styles.botao}
                  onPress={() => setContador(contador + 1)}
                >
                  <Text style={styles.textoBotao}>Aumentar</Text>
                </TouchableOpacity>
              </View>
            );
          };

          const styles = StyleSheet.create({
            container: {
              flex: 1,
              justifyContent: 'center',
              alignItems: 'center',
            },
            numero: {
              fontSize: 48,
              marginBottom: 10,
            },
            mensagem: {
              fontSize: 24,
              marginBottom: 20,
              color: '#34495e',
            },
            botao: {
              backgroundColor: '#3498db',
              padding: 15,
              borderRadius: 10,
            },
            textoBotao: {
              color: 'white',
              fontSize: 18,
            },
          });

Limpeza no useEffect

Às vezes precisamos "limpar" algo quando o componente some: 


            useEffect(() => {
              // Cria algo
              const timer = setTimeout(() => {
                console.log('Olá!');
              }, 1000);

              // Função de limpeza
              return () => {
                clearTimeout(timer); // Limpa o timer
              };
            }, []);

Atividade 3: Timer Simples

Crie um timer que:

  • 1. Mostre os segundos passando
  • 2. Tenha um botão para iniciar/pausar
  • 3. Limpe o timer quando pausar

Solução: Timer Simples


          import React, { useState, useEffect } from 'react-native';
          import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';

          const TimerSimples = () => {
            const [segundos, setSegundos] = useState(0);
            const [ativo, setAtivo] = useState(false);

            useEffect(() => {
              let timer = null;
              
              if (ativo) {
                timer = setInterval(() => {
                  setSegundos(s => s + 1);
                }, 1000);
              }

              // Limpa o timer quando desativa
              return () => {
                if (timer) clearInterval(timer);
              };
            }, [ativo]);

            const alternarTimer = () => setAtivo(!ativo);

            return (
              <View style={styles.container}>
                <Text style={styles.timer}>
                  {segundos} segundos
                </Text>
                <TouchableOpacity 
                  style={styles.botao}
                  onPress={alternarTimer}
                >
                  <Text style={styles.textoBotao}>
                    {ativo ? 'Pausar' : 'Iniciar'}
                  </Text>
                </TouchableOpacity>
              </View>
            );
          };

          const styles = StyleSheet.create({
            container: {
              flex: 1,
              justifyContent: 'center',
              alignItems: 'center',
            },
            timer: {
              fontSize: 36,
              marginBottom: 20,
            },
            botao: {
              backgroundColor: '#2ecc71',
              padding: 15,
              borderRadius: 10,
              minWidth: 120,
              alignItems: 'center',
            },
            textoBotao: {
              color: 'white',
              fontSize: 18,
            },
          });
